# Bolivar Reporte (Angular Admin)

Este proyecto es el módulo de reportería para el Club Bolívar, desarrollado en Angular. Incluye reportes detallados de membresías, KPIs y filtros avanzados.

## Características Principales

- **Reporte de Membresías**: Listado completo de socios con paginación, filtros y búsqueda.
- **Detalle de Pagos**: Modal interactivo para ver el desglose de cuotas pagadas y deudas.
- **UX Premium**: Diseño moderno con Bootstrap 5, iconos Feather y estilos personalizados.
- **Exportación**: Datos listos para ser exportados (funcionalidad base).
- **Responsive**: Adaptable a diferentes tamaños de pantalla.

## Instalación

1.  Clonar el repositorio:

    ```bash
    git clone https://github.com/TU_USUARIO/Bolivar-reporte.git
    cd Bolivar-reporte
    ```

2.  Instalar dependencias:

    ```bash
    npm install
    ```

3.  Ejecutar en desarrollo:
    ```bash
    npm start
    ```
    Visita `http://localhost:4200`

## Build para Producción

```bash
npm run build
```

Genera los archivos estáticos en la carpeta `dist/`.

## Tecnologías

- Angular 16+
- Bootstrap 5
- SASS/SCSS
- Feather Icons
